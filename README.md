LII Simposium Nacional de Alumbrado - App (ejemplo)

Instrucciones rápidas:

1. Descomprime este paquete en una carpeta.
2. En la carpeta, ejecuta:
   npm install
   npm start

3. Para desplegar en Vercel:
   - Conecta tu repositorio en Vercel y despliega (detecta React automáticamente).
   - O usar "npm run build" y subir la carpeta build.

Edición del programa:
- El archivo con las ponencias está en src/data/programa.json
- Actualiza títulos, ponentes, salas y horas allí.

Logo:
- public/assets/logo.png (reemplázalo si quieres)
